Mise en jambe (niveau 1)

  1. Le robot se trouve dans le coin sud-ouest, orienté vers le nord. Le faire avancer de deux cases.
  2. Le robot se trouve dans le coin sud-ouest, orienté vers le nord. Marquer 4 cases exactement.

Prise en compte de l'environnement (niveau 2)

  1. Le robot se trouve dans le coin sud-ouest, orientation inconnue. L'orienter vers le nord.
  2. Le robot se trouve n'importe où, orientation inconnue ; Avancer exactement 5 fois.
  3. Le robot se trouve n'importe où, orientation inconnue ; Marquer exactement 5 cases.

Utilisation du "Tant que" (niveau 3)

  1. Le robot se trouve dans le coin sud-ouest, orientation inconnue. L'orienter vers le nord. Déjà fait avec des instructions conditionnelles, mais peut se faire avec des instructions itératives.
  2. Le robot est n’importe où sur le terrain ; le faire avancer jusqu’au mur d’en face.
  3. Le robot se trouve dans le coin sud-ouest, orienté vers l’est. Lui faire faire le tour du terrain.
  4. Le robot est n’importe où sur le terrain ; Le faire avancer jusqu’au mur d’en face en marquant son chemin.
  5. Le robot se trouve dans le coin sud-ouest, orienté vers le nord. Aller jusqu’au coin nord-est.
  6. Le robot se trouve dans le coin sud-ouest, orienté vers le nord ; il y a du minerai sur le bord est. Trouver le minerai.
  7. Le robot se trouve dans le coin sud-ouest, orienté vers le nord. Faire un tour sur le bord, puis un tour une case à l’intérieur sans couper son chemin.
  8. Aller au milieu d’un bord
  9. Faire le tour du terrain et revenir au point de départ exactement.
  10. Le robot se trouve dans le coin sud-ouest, orienté vers le nord. Aller jusqu’au coin nord-ouest et retour.
  11. Le robot est quelque part sur le bord sud, orienté vers l’est. Faire exactement un tour.
  12. Aller au milieu d’un bord, sans laisser de marque.
  13. Le robot se trouve dans le coin sud-ouest, orienté vers le nord. Aller jusqu’au coin nord-ouest.
  14. Idem. Aller jusqu’au coin nord-est et retour.
  15. Idem. Faire le tour dans le sens inverse des aiguilles d’une montre.
  16. Le robot est n’importe où. Placer le dans un coin.
  17. Le robot se trouve dans le coin sud-ouest, orienté vers le nord ; il y a du minerai sur un bord. Trouver le minerai.
  18. Aller au centre du domaine
  19. Le robot est dans le coin sud-ouest, orienté vers le nord ; marquer tout le domaine
  20. Le robot est dans le coin sud-ouest, orienté vers le nord ; Il y a du minerai !
  21. Le robot est dans le coin sud-ouest, orienté vers le nord ; Il y a peut-être du minerai
  22. Le robot est n’importe où, orientation inconnue ; Il y a du minerai !
  23. Le robot est n’importe où, orientation inconnue ; Il y a peut-être du minerai
